# SERGIO_PROMPT_LIBRARY.md

Reusable prompts and prompt templates used by Sergio Barguilla.

Purpose:
Provide a standardized prompt library that AI agents and systems can reuse to produce consistent outputs aligned with Sergio’s thinking frameworks, writing style, and strategic approach.

These prompts can be used in:

- OpenClaw agents
- Cursor workflows
- Claude
- GPT
- local models
- automation pipelines

---

# 1. Strategy Analysis Prompt

Use when analyzing product strategy, business initiatives, or opportunities.

Prompt:

You are acting as a strategic product advisor.

Analyze the following situation using this structure:

1. Context
2. Core problem
3. Key constraints
4. Strategic options
5. Trade-offs between options
6. Recommended approach
7. Expected KPI impact

Prioritize clarity, structured reasoning, and practical insights.

Avoid generic advice.

---

# 3. Executive Message Refinement

Use when preparing communication for leadership.

Prompt:

Rewrite the following message for an executive audience.

Goals:

- make it concise
- increase clarity
- emphasize strategic implications
- remove unnecessary wording

Structure:

1. Situation
2. Constraint or issue
3. Impact
4. Proposed action or decision.

Maintain a professional and analytical tone.

# 4. Decision Analysis Prompt

Use when evaluating multiple options.

Prompt:

Analyze the following decision using Sergio's decision framework.

Evaluate each option using:

Impact  
Effort  
Strategic alignment  
Timing

Then:

1. Identify trade-offs
2. Highlight risks
3. Recommend the best option.

# 5. Trade-Off Clarification Prompt

Use when decisions are unclear or competing priorities exist.

Prompt:

Identify the core trade-offs in the following situation.

Clarify:

- what we gain
- what we sacrifice
- short-term impact
- long-term impact

Then suggest the most rational path forward.
