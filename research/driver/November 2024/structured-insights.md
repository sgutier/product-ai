# Structured Insights — Driver Persona (Mobile Track & Trace Study)

## Persona
Driver

## Market
Transportation companies (various sizes: 1 truck to 51+ trucks), primarily transport and freight forwarding

## Research Type
Quantitative survey study

## Sample Size
73 drivers

---

## Core Jobs

- Navigate efficiently using mobile navigation tools
- Communicate with dispatchers/owners via phone calls and messaging apps
- Complete transport orders with proper documentation
- Use tachographs for compliance and tracking
- Manage work tasks using personal or company-provided mobile devices
- Share location information when necessary for work coordination
- Find fuel prices and plan refueling stops
- Refer useful tools to company when appropriate

---

## Pain Points

- **App stability issues**: Crashes, especially in busy areas (e.g., Birmingham)
- **Navigation accuracy problems**: Issues with tonnage ban zones, ADR (dangerous goods) routing
- **Lack of company-provided roaming**: 67% don't receive international roaming data packages
- **Privacy concerns**: Reluctance to share location on private phones, prefer work-only tracking
- **Data usage worries**: Concern about location sharing consuming mobile data packages
- **Missing features**: Inability to plan routes on PC, poor search capabilities
- **Distraction concerns**: Some drivers worry about fuel price visibility causing distraction
- **Lack of route diversion features**: Need better handling of road closures, low bridges, weight restrictions
- **Offline navigation limitations**: Need improved offline navigation capabilities
- **Customer service issues**: Requests for better customer service

---

## Frictions

- Using private phones for work (75% of drivers) without company support
- Switching between multiple tools (navigation, tachograph, communicators, tablets)
- Lack of integrated solutions combining navigation, communication, and tracking
- App crashes disrupting work flow
- Inaccurate navigation causing routing problems
- Privacy concerns when sharing location on personal devices
- Data usage concerns when using location sharing features
- Pre-planned refueling making real-time fuel prices less useful
- Missing features from previous app versions (e.g., "Road Lords")

---

## Monetization Signals

- Willingness to refer navigation software for practical rewards (equipment, automotive accessories)
- Preference for premium navigation trial licenses as referral rewards
- Interest in features that enhance work experience (comfort equipment, automotive accessories)
- Value placed on reliable, stable navigation tools
- Recognition that good navigation tools improve work efficiency
- Interest in shopping vouchers and flexible reward options

---

## Behavioral Signals

- High reliance on mobile phone navigation (55 out of 73 drivers use daily)
- Extensive use of personal mobile phones for work (75% use private phones)
- Frequent use of communicators (WhatsApp, Viber, etc.) for work communication
- Preference for phone calls over other communication methods
- Experience with location sharing apps (79% have used)
- Familiarity with existing tracking systems (vehicle GPS, office tracking)
- Use of multiple devices (mobile phone, tablet, laptop, external navigation)
- Preference for work-focused features over personal features
- Desire for control over who can see location information

---

## Emotional Signals

- Frustration with app crashes and stability issues
- Concern about privacy when using personal phones for work tracking
- Skepticism about continuous tracking features
- Preference for alerts only when vehicle is in danger
- Positive sentiment toward fuel price visibility (51% think "That would be great!")
- Concern about distraction from additional features
- Nostalgia for previous app versions with better features
- Trust issues with app accuracy ("If it's as accurate as the rest of the app I wouldn't dare to rely on it")
- Relief when tools work reliably without crashes
- Appreciation for practical, work-enhancing rewards
